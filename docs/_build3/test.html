1. Installation

pip install snowmobile

2. Export snowmobile.toml 

import snowmobile
from pathlib import Path

snowmobile.Configuration(export_dir=Path.cwd())

Provide a target directory to the export_dir argument of snowmobile.Configuration to export a default snowmobile.toml file; the above snippet exports to the current working directory.
